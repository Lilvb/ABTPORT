# Abanob Thabet — Portfolio

A personal portfolio site built with React, GSAP, and Tailwind CSS — featuring smooth scroll, custom animations, and a dark/light theme toggle.

## Features

- Animated gooey blob background using GSAP + SVG filters
- Custom cursor with hover states
- Typing effect intro animation
- Smooth scrolling powered by Lenis
- Dark / light theme toggle with persistence
- Fully responsive with reduced-motion support

---

## Tech Stack

| Category | Tools |
|---|---|
| Framework | React + TypeScript + Vite |
| Styling | Tailwind CSS |
| Animation | GSAP, Lenis |
| Icons | Lucide React |

## Structure

```text
src/
├── components/   # Reusable UI pieces (cards, cursor, blob, etc.)
├── sections/     # Page sections (Hero, About, Projects, Contact...)
├── hooks/        # Custom hooks (theme, scroll, typing effect)
└── context/      # Theme context
```
